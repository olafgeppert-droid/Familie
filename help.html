Hilfe & Anleitung 📖

Wichtig: Personen- und Ring-Code werden automatisch vom Programm vergeben. Die Logik wurde vom Wappenspender, Olaf Geppert, erfunden und vorgegeben.

🧭 Zweck der Anwendung

Diese Anwendung hilft dir, Familienmitglieder zu erfassen, den grafischen Stammbaum zu visualisieren und die Vererbung der Wappenringe der Familie GEPPERT nachzuverfolgen. Du hast eine Tabelle, eine Baumansicht, Export/Import, Druck und Statistik.

🧬 Personen-Code-System

  • Der Personen-Code wird vom System vergeben (keine manuelle Eingabe).
  • Stammvater = 1, Partner(in) = 1x (x immer klein).
  • Kinder des Stammvaters: 1A, 1B, 1C … (Buchstaben stets groß).
  • Enkel: Zahlen hinter dem Code des jeweiligen Kindes in Geburtsreihenfolge (TT.MM.JJJJ). Beispiel: erstes Kind von 1A → 1A1, zweites → 1A2.
  • Buchstaben werden automatisch groß, das Partner-Suffix x bleibt klein.

💍 Ring-Codes

  • Ring-Code entspricht dem Personen-Code.
  • Bei Vererbung: Anschlussgravur mit Pfeil, z. B. 1B → 1B2.
  • Partner tragen ihr x im Ring-Code (z. B. 1x).
  • So bleibt der Ring des Erblassers vollständig erhalten.

🗂️ Daten & Bedienung

  • Neue Person: Dialog öffnen; Abbrechen geht jederzeit.
  • Bearbeiten: Doppelklick auf Tabellenzeile.
  • Suche: Filtert die Tabelle live; Fundstellen werden markiert.
  • Person löschen: per Name oder Code; Referenzen werden entfernt.
  • Undo/Redo: Änderungen rückgängig/erneut anwenden.

🌳 Stammbaum

  • Automatisches Neu-Rendern bei jeder Datenänderung (auch nach Import).
  • Generationen vertikal; Partner-Buslinie (wenn keine gemeinsamen Kinder).
  • Eltern–Kind: Senkrecht nach unten zur gemeinsamen Linie, dann zu den Kindern.
  • Knoten: Zeile 1 „Code / Name“, Zeile 2 „Generation: n / Geburtsdatum“; farbig je Generation.

📊 Statistik

  • Gesamt Anzahl Personen
  • Geschlecht männlich, weiblich, divers
  • Generationen pro Generation

💾 Export / Import

  • Export: JSON und CSV. Auf iOS → „Teilen“, sonst Download.
  • Import: JSON/CSV; Baum rendert danach automatisch neu.
  • Drucken: Auswahl (Tabelle/Baum/Beides); erzeugt PDF über den Druckdialog.
  • Daten sind nur lokal gespeichert → bitte regelmäßig sichern.
